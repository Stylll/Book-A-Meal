Book A Meal App

API Endpoint
https://bookamealapp.herokuapp.com/api/v1

Book-A-Meal is an application that allows customers to make food orders and helps the food vendor know what the customers what to eat.

POST/users/signup

In Book-A-Meal, a user is required to create an account in order to access other api endpoints. You can decide to signup as a customer or as a caterer. You get an access token once your account is successfully created.

Sign in to your account
POST/users/signin

You are required to sign in using your email and password in order to get a generated token. This token should be passed in the header for every other request made to the api for authentication and authorization.

Request for Password reset token
PUT/users/forgotpassword

As a registered user, you can request for a password reset token link. This link will be set to your email and will expire after 1 hour.
